ESP32 RF Sniffer & Transmitter (ESP-IDF)This project uses the rc-switch library port for ESP-IDF to control WL101/WL102 modules.WiringRX Data -> GPIO 18TX Data -> GPIO 19VCC -> 3.3VGND -> GNDHow to Build & FlashInitialize the project:Ensure you have the ESP-IDF environment set up.Build:Run the following command. This will automatically download the nopnop2002/esp-idf-rc-switch component defined in idf_component.yml.idf.py build
Flash:Connect your ESP32 and run:idf.py -p (PORT) flash monitor
(Replace (PORT) with your actual serial port, e.g., COM3 or /dev/ttyUSB0)UsagePoint your motorized bed remote at the receiver.Press a button.Look at the serial monitor. You should see the Decimal, Protocol, and BitLength.Copy those numbers into the mySwitch->send(...) function in main.c to clone the signal.
